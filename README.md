# RoboMo 2.0 Capstone Submission

This repository contains the full capstone submission for RoboMo 2.0. It is organized into two submission parts (Phase A and Phase B) plus the complete project codebase. The codebase includes its own detailed README with setup and deployment instructions.

## Repository Structure

- `Phase A/`
- `Phase B/`
- `Codebase/RoboMo-2.0/`

## Part A (Phase A)

Located in `Phase A/`:

- `Capstone_Phase-A_25-2-D-11.docx`
- `Capstone_Phase-A_25-2-D-11.pptx`

These files correspond to the Phase A deliverables for the capstone project.

## Part B (Phase B)

Located in `Phase B/`:

- `Capstone_Phase-B_25-2-D-11.docx`
- `Poster_RoboMo2_25-2-D-11.pdf`
- `Video_Capstone_Phase-B_25-2-D-11.mp4`

These files correspond to the Phase B deliverables for the capstone project.

## Codebase

The full application source is located in `Codebase/RoboMo-2.0/`. It includes:

- `client/`: React web dashboard
- `server/`: Node.js backend with API and WebSocket streaming
- `ml/`: Python ML service (FastAPI) plus training and artifacts
- `standards_baseline/`: Baseline rules/engine for device standards
- `vpn/`: OpenVPN configuration

For setup, running, and deployment details, see `Codebase/RoboMo-2.0/README.md`.
